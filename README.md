# ⚔️ МАЙНБАТЛ - Битва стратегий на реальном сервере

## 🌐 Сервер: `188.120.254.248:25565`

### 📁 Структура
- `глава.py` - главный менеджер
- `стратегии/` - папка с вашими стратегиями

### 🚀 Как запустить
1. Запустите `bridge.js`
2. Запустите главу: `python глава.py`

### ✍️ Как создать свою стратегию
1. Создайте файл в папке `стратегии/`
2. Определите функцию `стратегия(карта)`
3. Функция должна возвращать решение
